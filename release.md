### üöÄ About This Release

We are pleased to announce the **initial pre-release** of **Glance**, a professional, high-performance telemetry dashboard designed for real-time data visualization and analysis.

### ‚ú® Key Features

*   **Real-Time Visualization**: High-frequency plotting for time-series data.
*   **GPS Tracking**: Integrated map visualization for live location tracking.
*   **Raw Data Inspection**: Built-in raw telemetry packet viewer for low-level debugging.
*   **Session Management**: Save and load custom dashboard layouts and project configurations.
*   **Data Logging**: Automated logging to CSV and JSON with summary generation.
*   **Customizable UI**: Drag-and-drop widget arrangement with a modern, dark-themed interface.
*   **Cross-Platform**: Fully supported on Windows and Linux.
